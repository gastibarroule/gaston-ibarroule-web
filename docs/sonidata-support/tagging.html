Tagging & Metadata

The Naming View is the core of Sonidata. It's where you assign UCS-compliant metadata to your recordings using the industry-standard Universal Category System (UCS v8.2).

UCS Filename Format

{CatID}{UserCategory}_{VendorCategory}{FXName}_{CreatorID}_{SourceID}_{Location-UserData}

Example: ANMLDog_Bark-Close_GI_MKH8060_Park.wav

Metadata Fields

FieldDescriptionExample
CategoryPrimary UCS categoryANIMALS
SubcategorySpecific sub-typeDOG
CatIDShort UCS code (auto-filled)ANML
FX NameDescriptive name for the soundBark-Close
Creator IDYour initials or identifierGI
Source IDMicrophone or source usedMKH8060
LocationWhere the sound was recordedPark
User CategoryCustom category tag (optional)‚Äî
Vendor CategoryVendor-specific prefix (optional)‚Äî
User DataAny extra info (optional)‚Äî

How to Tag a Recording

  1. 1Search for a category using SmartSearch ‚Äî type "dog" and it finds ANIMALS > DOG even if you don't know the exact name.
  2. 2Select from the dual-wheel picker (Category on left, Subcategory on right).
  3. 3Fill in the FX Name, or use Voice Slate to dictate it.
  4. 4Add Creator ID and Source ID ‚Äî enable "Remember IDs" in Settings to save across sessions.
  5. 5Add Location ‚Äî tap the location button to auto-fill from GPS, or type manually.
  6. 6Attach a Photo if desired (tap the camera icon).
  7. 7Preview the filename in the bottom bar, then tap "Rename File".

SmartSearch

The category search uses Apple's Natural Language framework for semantic understanding ‚Äî not just text matching. Type "rain" to find WEATHER > RAIN and related categories. Synonym matching means "puppy" finds ANIMALS > DOG. Exact matches score highest, followed by subcategory, synonyms, and semantic neighbors.

Batch Rename

Rename multiple files at once: enter Edit Mode, select files, tap "Rename". All selected files receive the same metadata and are renamed sequentially with unique indices.
